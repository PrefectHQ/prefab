Column arranges children vertically. Use `gap` to control spacing between items.

## Basic Usage

```python Python
from prefab_ui.components import (
    Column,
    Div,

with Column(gap=4):

    Div(css_class="bg-emerald-500 rounded-md h-10")
```

## Alignment

Use `align` to control cross-axis alignment (horizontal, for Column) and `justify` to control distribution along the main axis (vertical).

<ComponentPreview auto json={`{"cssClass":"gap-4 items-center","type":"Column","children":[{"cssClass":"bg-emerald-500 rounded-md h-10 w-full","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10 w-3/4","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10 w-1/2","type":"Div"}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Div,
)

with Column(gap=4, align="center"):
    Div(css_class="bg-emerald-500 rounded-md h-10 w-full")
    Div(css_class="bg-emerald-500 rounded-md h-10 w-3/4")
    Div(css_class="bg-emerald-500 rounded-md h-10 w-1/2")
```

## API Reference

<Card icon="code" title="Column Parameters">
<ParamField body="gap" type="int | tuple | None" default="None">
  Gap between children. An int maps to `gap-{n}`. A tuple `(x, y)` sets per-axis gaps; use `None` to skip an axis.
</ParamField>

<ParamField body="align" type="str | None" default="None">
  Cross-axis alignment: `"start"`, `"center"`, `"end"`, `"stretch"`, `"baseline"`.
</ParamField>

<ParamField body="justify" type="str | None" default="None">
  Main-axis distribution: `"start"`, `"center"`, `"end"`, `"between"`, `"around"`, `"evenly"`, `"stretch"`.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
