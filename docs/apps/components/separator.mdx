## Usage

Separators create visual divisions between content areas.

<ComponentPreview auto json={`{"type":"Column","children":[{"content":"Content above the separator","type":"P"},{"cssClass":"my-4","type":"Separator","orientation":"horizontal"},{"content":"Content below the separator","type":"P"}]}`} />

```python Separator Example
from prefab_ui.components import (
    Column,
    P,
    Separator,
)

with Column():
    P("Content above the separator")
    Separator(css_class="my-4")
    P("Content below the separator")
```

## Vertical Orientation

<ComponentPreview auto json={`{"cssClass":"items-center","type":"Row","gap":4,"children":[{"type":"Span","content":"Left"},{"cssClass":"h-4","type":"Separator","orientation":"vertical"},{"type":"Span","content":"Middle"},{"cssClass":"h-4","type":"Separator","orientation":"vertical"},{"type":"Span","content":"Right"}]}`} />

```python Vertical Separator
from prefab_ui.components import (
    Row,
    Separator,
    Span,
)

with Row(gap=4, css_class="items-center"):
    Span("Left")
    Separator(orientation="vertical", css_class="h-4")
    Span("Middle")
    Separator(orientation="vertical", css_class="h-4")
    Span("Right")
```

## In Card Content

<ComponentPreview auto json={`{"type":"Card","children":[{"type":"CardContent","children":[{"content":"First section","type":"P"},{"cssClass":"my-4","type":"Separator","orientation":"horizontal"},{"content":"Second section","type":"P"},{"cssClass":"my-4","type":"Separator","orientation":"horizontal"},{"content":"Third section","type":"P"}]}]}`} />

```python Separator in Card
from prefab_ui.components import (
    Card,
    CardContent,
    P,
    Separator,
)

with Card():
    with CardContent():
        P("First section")
        Separator(css_class="my-4")
        P("Second section")
        Separator(css_class="my-4")
        P("Third section")
```
