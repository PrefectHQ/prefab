---
title: ChartSeries
description: JSON Schema for the ChartSeries component.
---

Series definition for cartesian charts (Bar, Line, Area).

```json
{
  "type": "object",
  "properties": {
    "dataKey": {
      "description": "Data field to plot",
      "type": "string"
    },
    "label": {
      "type": [
        "string",
        "null"
      ],
      "default": null,
      "description": "Display label (defaults to dataKey)"
    },
    "color": {
      "type": [
        "string",
        "null"
      ],
      "default": null,
      "description": "CSS color override"
    }
  },
  "required": [
    "dataKey"
  ]
}
```
