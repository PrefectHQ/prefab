## Div

A block container with no default styling. Style it entirely through `css_class`:

<ComponentPreview auto json={`{"view":{"cssClass":"flex items-center gap-3 rounded-lg border p-4","type":"Div","children":[{"cssClass":"bg-emerald-500 rounded-md h-8 w-8","type":"Div"},{"content":"Custom layout with Div","type":"P"}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Div,
    P,
)

with Div(css_class="flex items-center gap-3 rounded-lg border p-4"):
    Div(css_class="bg-emerald-500 rounded-md h-8 w-8")
    P("Custom layout with Div")
```

## Span

An inline text element with no default styling. Useful for inline annotations or colored text:

<ComponentPreview auto json={`{"view":{"cssClass":"gap-2 items-baseline","type":"Row","children":[{"content":"Status:","type":"P"},{"cssClass":"text-sm text-success font-medium","type":"Span","content":"Online"}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    P,
    Row,
    Span,
)

with Row(gap=2, align="baseline"):
    P("Status:")
    Span("Online", css_class="text-sm text-success font-medium")
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "cssClass": "gap-2 items-baseline",
    "type": "Row",
    "children": [
      {
        "content": "Status:",
        "type": "P"
      },
      {
        "cssClass": "text-sm text-success font-medium",
        "type": "Span",
        "content": "Online"
      }
    ]
  }
}
```
</CodeGroup>
</ComponentPreview>
