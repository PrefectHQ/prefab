---
title: Icon
description: JSON Schema for the Icon component.
---

Renders a lucide icon by name.

```json
{
  "type": "object",
  "properties": {
    "type": {
      "const": "Icon",
      "default": "Icon",
      "type": "string"
    },
    "cssClass": {
      "type": [
        "string",
        "null"
      ],
      "default": null,
      "description": "CSS/Tailwind classes for styling"
    },
    "name": {
      "description": "Lucide icon name in kebab-case",
      "type": "string"
    },
    "size": {
      "default": "default",
      "description": "Icon size variant",
      "enum": [
        "sm",
        "default",
        "lg"
      ],
      "type": "string"
    }
  },
  "required": [
    "type",
    "name"
  ]
}
```
