Radio buttons let users select exactly one option from a set of choices.

## Basic Usage

<ComponentPreview auto json={`{"$protocol":"prefab","view":{"type":"RadioGroup","name":"size","children":[{"type":"Radio","value":"sm","label":"Small","checked":false,"disabled":false,"required":false},{"type":"Radio","value":"md","label":"Medium","checked":true,"disabled":false,"required":false},{"type":"Radio","value":"lg","label":"Large","checked":false,"disabled":false,"required":false}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Radio,
    RadioGroup,
)

with RadioGroup(name="size"):
    Radio(value="sm", label="Small")
    Radio(value="md", label="Medium", checked=True)
    Radio(value="lg", label="Large")
```

## Without RadioGroup

Individual radio buttons can be used independently:

<ComponentPreview auto json={`{"$protocol":"prefab","view":{"cssClass":"gap-3","type":"Column","children":[{"type":"Radio","value":"yes","label":"Yes","checked":false,"name":"answer","disabled":false,"required":false},{"type":"Radio","value":"no","label":"No","checked":true,"name":"answer","disabled":false,"required":false}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Radio,
)

with Column(gap=3):
    Radio(value="yes", label="Yes", name="answer")
    Radio(value="no", label="No", name="answer", checked=True)
```
```json Protocol icon="brackets-curly"
{
  "$protocol": "prefab",
  "view": {
    "cssClass": "gap-3",
    "type": "Column",
    "children": [
      {
        "type": "Radio",
        "value": "yes",
        "label": "Yes",
        "checked": false,
        "name": "answer",
        "disabled": false,
        "required": false
      },
      {
        "type": "Radio",
        "value": "no",
        "label": "No",
        "checked": true,
        "name": "answer",
        "disabled": false,
        "required": false
      }
    ]
  }
}
```
</CodeGroup>

## API Reference

<Card icon="code" title="RadioGroup Parameters">
<ParamField body="name" type="str | None" default="None">
  Form field name shared by all radio buttons in the group.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>

<Card icon="code" title="Radio Parameters">
<ParamField body="value" type="str" required>
  Form value for this option.
</ParamField>

<ParamField body="label" type="str | None" default="None">
  Label text displayed next to the radio button.
</ParamField>

<ParamField body="checked" type="bool" default="False">
  Whether this radio button is initially selected.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  Form field name. Usually inherited from RadioGroup.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the radio button is non-interactive.
</ParamField>

<ParamField body="required" type="bool" default="False">
  Whether a selection is required for form submission.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
