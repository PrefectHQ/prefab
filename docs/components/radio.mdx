Radio buttons let users select exactly one option from a set of choices.

## Basic Usage

<ComponentPreview json={{"view":{"name":"size","type":"RadioGroup","children":[{"name":"radio_1","type":"Radio","value":"sm","label":"Small","checked":false,"disabled":false,"required":false},{"name":"radio_2","type":"Radio","value":"md","label":"Medium","checked":true,"disabled":false,"required":false},{"name":"radio_3","type":"Radio","value":"lg","label":"Large","checked":false,"disabled":false,"required":false}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Radio,
    RadioGroup,
)

with RadioGroup(name="size"):
    Radio(value="sm", label="Small")
    Radio(value="md", label="Medium", checked=True)
    Radio(value="lg", label="Large")
```

## Without RadioGroup

## With Actions

The `on_change` parameter on `RadioGroup` runs actions when the selection changes. `$event` is the `value` string of the selected radio:

<ComponentPreview json={{"view":{"cssClass":"gap-4","type":"Column","children":[{"name":"theme","type":"RadioGroup","onChange":{"action":"setState","key":"choice","value":"You picked: {{ $event }}"},"children":[{"name":"radio_4","type":"Radio","value":"light","label":"Light","checked":false,"disabled":false,"required":false},{"name":"radio_5","type":"Radio","value":"dark","label":"Dark","checked":false,"disabled":false,"required":false},{"name":"radio_6","type":"Radio","value":"system","label":"System","checked":true,"disabled":false,"required":false}]},{"cssClass":"text-muted-foreground","content":"{{ choice | default:'Make a selection' }}","type":"Text"}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Column, Radio, RadioGroup, Text
from prefab_ui.actions import SetState
from prefab_ui.rx import Rx

choice = Rx("choice")

with Column(gap=4):
    with RadioGroup(
        name="theme",
        on_change=SetState("choice", f"You picked: {Rx('$event')}"),
    ):
        Radio(value="light", label="Light")
        Radio(value="dark", label="Dark")
        Radio(value="system", label="System", checked=True)
    Text(
        choice.default("Make a selection"),
        css_class="text-muted-foreground",
    )
```
```json Protocol icon="brackets-curly"

    "children": [

        "name": "theme",
```

## API Reference

<Card icon="code" title="RadioGroup Parameters">
<ParamField body="name" type="str | None" default="None">
  Form field name shared by all radio buttons in the group.
</ParamField>

<ParamField body="on_change" type="Action | list[Action] | None" default="None">
  Action(s) to execute when the selection changes. `$event` is the `value` string of the selected radio.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>

<Card icon="code" title="Radio Parameters">
<ParamField body="value" type="str" required>
  Form value for this option.
</ParamField>

<ParamField body="label" type="str | None" default="None">
  Label text displayed next to the radio button.
</ParamField>

<ParamField body="checked" type="bool" default="False">
  Whether this radio button is initially selected.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  Form field name. Usually inherited from RadioGroup.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the radio button is non-interactive.
</ParamField>

<ParamField body="required" type="bool" default="False">
  Whether a selection is required for form submission.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
