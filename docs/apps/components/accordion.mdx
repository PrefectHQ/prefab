Accordions let users expand and collapse sections to reveal content progressively. Each `AccordionItem` has a clickable title and hidden content that appears on expand.

```python Python
from prefab_ui.components import Accordion, AccordionItem, Text

with Accordion(default_value="item-1"):
    with AccordionItem("Is it accessible?", value="item-1"):
        Text("Yes. It adheres to the WAI-ARIA design pattern.")
```

## Multiple Open

Set `accordion_type="multiple"` to allow more than one section open at a time.

<ComponentPreview auto json={`{"type":"Accordion","accordionType":"multiple","collapsible":true,"defaultValue":["s1","s2"],"children":[{"type":"AccordionItem","title":"Section One","value":"s1","children":[{"type":"Text","content":"First section content."}]},{"type":"AccordionItem","title":"Section Two","value":"s2","children":[{"type":"Text","content":"Second section content."}]},{"type":"AccordionItem","title":"Section Three","value":"s3","children":[{"type":"Text","content":"Third section content."}]}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import Accordion, AccordionItem, Text

with Accordion(accordion_type="multiple", default_value=["s1", "s2"]):
    with AccordionItem("Section One", value="s1"):
        Text("First section content.")
    with AccordionItem("Section Two", value="s2"):
        Text("Second section content.")
    with AccordionItem("Section Three", value="s3"):
        Text("Third section content.")
```

## API Reference

<Card icon="code" title="Accordion Parameters">
<ParamField body="accordion_type" type="str" default="single">
  `"single"` allows one item open at a time; `"multiple"` allows several.
</ParamField>

<ParamField body="collapsible" type="bool" default="True">
  Whether the open item can be collapsed (single mode only).
</ParamField>

<ParamField body="default_value" type="str | list[str] | None" default="None">
  Initially expanded item(s). Pass a string for single mode, a list for multiple.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>

<Card icon="code" title="AccordionItem Parameters">
<ParamField body="title" type="str" required>
  Trigger label text. Can be passed as a positional argument.
</ParamField>

<ParamField body="value" type="str | None" default="None">
  Unique value identifying this item. Defaults to the title.
</ParamField>
</Card>
