Row arranges children horizontally with flexbox. Use `gap` to control spacing between items.

## Basic Usage

<ComponentPreview auto json={`{"cssClass":"gap-4","type":"Row","children":[{"cssClass":"bg-emerald-500 rounded-md h-10 w-20","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10 w-20","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10 w-20","type":"Div"}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Div,
    Row,
)

with Row(gap=4):
    Div(css_class="bg-emerald-500 rounded-md h-10 w-20")
    Div(css_class="bg-emerald-500 rounded-md h-10 w-20")
    Div(css_class="bg-emerald-500 rounded-md h-10 w-20")
```

## API Reference

<Card icon="code" title="Row Parameters">
<ParamField body="gap" type="int | tuple | None" default="None">
  Gap between children. An int maps to `gap-{n}`. A tuple `(x, y)` sets per-axis gaps; use `None` to skip an axis.
</ParamField>

<ParamField body="align" type="str | None" default="None">
  Cross-axis alignment: `"start"`, `"center"`, `"end"`, `"stretch"`, `"baseline"`.
</ParamField>

<ParamField body="justify" type="str | None" default="None">
  Main-axis distribution: `"start"`, `"center"`, `"end"`, `"between"`, `"around"`, `"evenly"`, `"stretch"`.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
