Sliders let users select a numeric value from a range by dragging a handle. The default range is 0–100.

## Basic Usage

<ComponentPreview json={{"view":{"type":"Slider","min":0,"max":100,"value":50.0,"name":"slider_2","disabled":false},"state":{"slider_2":50.0}}}>

## Displaying the Value

Create the slider with `defer=True` to get a reference to it, then use `.rx` to bind its value into a label. Call `insert()` to place the slider in the tree:

<ComponentPreview json={{"view":{"cssClass":"gap-2","type":"Column","children":[{"cssClass":"justify-between","type":"Row","children":[{"type":"Label","text":"Volume"},{"content":"{{ slider_3 | number }}%","bold":true,"type":"Text"}]},{"type":"Slider","min":0,"max":100,"value":75.0,"name":"slider_3","disabled":false}]},"state":{"slider_3":75.0}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Label,
    Row,
    Slider,
    Text,
    insert,
)

volume = Slider(value=75, defer=True)

with Column(gap=2):
    with Row(css_class="justify-between"):
        Label("Volume")
        Text(f"{volume.rx.number()}%", bold=True)
    insert(volume)
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "cssClass": "gap-2",
    "type": "Column",
    "children": [
      {
        "cssClass": "justify-between",
        "type": "Row",
        "children": [
          {"type": "Label", "text": "Volume"},
          {"content": "{{ slider_3 | number }}%", "bold": true, "type": "Text"}
        ]
      },
      {
        "type": "Slider",
        "min": 0,
        "max": 100,
        "value": 75.0,
        "name": "slider_3",
        "disabled": false
      }
    ]
  },
  "state": {"slider_3": 75.0}
}
```
</CodeGroup>
</ComponentPreview>

The `.number()` pipe strips trailing decimals so the display stays clean. `defer=True` prevents the slider from auto-attaching, letting you reference `.rx` before placing it with `insert()`.

## Range and Step

Use `min`, `max`, and `step` to constrain the slider. Here the range is 0–5 with half-step increments:

<ComponentPreview json={{"view":{"cssClass":"gap-2","type":"Column","children":[{"cssClass":"justify-between","type":"Row","children":[{"type":"Label","text":"Rating"},{"content":"{{ rating }}","bold":true,"type":"Text"}]},{"type":"Slider","min":0.0,"max":5.0,"value":4.0,"step":0.5,"name":"rating","disabled":false}]},"state":{"rating":4.0}}}>

from prefab_ui.components import (

rating = Slider(value=4, name="rating", min=0, max=5, step=0.5, defer=True)

## With Actions

The `on_change` parameter runs actions when the slider value changes. `$event` is the current numeric value:

<ComponentPreview json={{"view":{"cssClass":"gap-4","type":"Column","children":[{"type":"Slider","min":0,"max":100,"value":50.0,"name":"slider_4","disabled":false,"onChange":{"action":"setState","key":"level","value":"{{ $event }}"}},{"cssClass":"text-muted-foreground","content":"{{ level | default:'Drag the slider' }}","type":"Text"}]},"state":{"slider_4":50.0}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.actions import SetState
from prefab_ui.components import Column, Rx, Slider, Text

level = Rx("level")

with Column(gap=4):
    Slider(
        value=50,
        on_change=SetState("level", Rx("$event")),
    )
    Text(
        level.default("Drag the slider"),
        css_class="text-muted-foreground",
    )
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "cssClass": "gap-4",
    "type": "Column",
    "children": [
      {
        "type": "Slider",
        "min": 0,
        "max": 100,
        "value": 50.0,
        "name": "slider_4",
        "disabled": false,
        "onChange": {"action": "setState", "key": "level", "value": "{{ $event }}"}
      },
      {
        "cssClass": "text-muted-foreground",
        "content": "{{ level | default:'Drag the slider' }}",
        "type": "Text"
      }
    ]
  },
  "state": {"slider_4": 50.0}
}
```
</CodeGroup>
</ComponentPreview>

<Tip>
For simply displaying the slider's current value, [`.rx` binding](#displaying-the-value) is usually easier. Use `on_change` when you need to run custom logic like triggering a tool call or updating multiple state keys.
</Tip>

## API Reference

<Card icon="code" title="Slider Parameters">

<ParamField body="value" type="float | None" default="None">
  Initial value of the slider.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  State key for the slider's current value. Set this to bind the slider to state for display or form submission.
</ParamField>

<ParamField body="min" type="float" default="0">
  Minimum value of the range.
</ParamField>

<ParamField body="max" type="float" default="100">
  Maximum value of the range.
</ParamField>

<ParamField body="step" type="float | None" default="None">
  Step increment. If not provided, the slider moves continuously.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the slider is non-interactive.
</ParamField>

<ParamField body="variant" type="str" default="default">
  Visual variant for the filled track: `"default"`, `"success"`, `"warning"`, `"destructive"`, `"info"`, `"muted"`.
</ParamField>

<ParamField body="indicator_class" type="str | None" default="None">
  Tailwind classes for the filled track. Overrides variant coloring.
</ParamField>

<ParamField body="orientation" type="str" default="horizontal">
  Layout direction: `"horizontal"` or `"vertical"`. Vertical sliders fill their container's height.
</ParamField>

<ParamField body="handle_style" type="str" default="circle">
  Thumb shape: `"circle"` (default round) or `"bar"` (tall rounded rectangle).
</ParamField>

<ParamField body="on_change" type="Action | list[Action] | None" default="None">
  Action(s) to execute when the value changes. `$event` is the current numeric value.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>
