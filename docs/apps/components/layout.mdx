Layout containers control how child components are arranged. They use Python context managers for nesting — anything created inside a `with` block becomes a child of that container.

## Row

Arranges children horizontally with flexbox. Use `gap` to control spacing between items.

```python Python
from prefab_ui.components import (
    Div,
    Row,

with Row(gap=4):
```

## Grid

CSS grid with a configurable number of columns (1-12). Children fill cells left-to-right, wrapping to new rows.

<ComponentPreview auto json={`{"type":"Grid","columns":3,"gap":4,"children":[{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"},{"cssClass":"bg-emerald-500 rounded-md h-10","type":"Div"}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Div,
```

## Div

A bare container with no default styling — the escape hatch for when Row, Column, and Grid don't express what you need. Style it entirely through `css_class`:

<ComponentPreview auto json={`{"cssClass":"flex items-center gap-3 rounded-lg border p-4","type":"Div","children":[{"cssClass":"bg-emerald-500 rounded-md h-8 w-8","type":"Div"},{"content":"Custom layout with Div","type":"P"}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Div,
    P,
)

with Div(css_class="flex items-center gap-3 rounded-lg border p-4"):
    Div(css_class="bg-emerald-500 rounded-md h-8 w-8")
    P("Custom layout with Div")
```

## Span

An inline text element with no default styling. Useful for inline annotations or colored text:

<ComponentPreview auto json={`{"cssClass":"items-baseline","type":"Row","gap":2,"children":[{"content":"Status:","type":"P"},{"cssClass":"text-sm text-success font-medium","type":"Span","content":"Online"}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    P,
    Row,
    Span,
)

with Row(gap=2, css_class="items-baseline"):
    P("Status:")
    Span("Online", css_class="text-sm text-success font-medium")
```

## Nesting

Containers nest arbitrarily — a Grid can hold Columns, a Row can sit inside a Column, and so on. The context manager syntax makes the hierarchy visible:

```python Nested Layout
from prefab_ui.components import (
    Button,
    Column,
    Grid,
    H3,
    P,
    Row,
)

with Grid(2, gap=6):
    with Column():
        H3("Details")
        P("Some information here.")
    with Column():
        H3("Actions")
        with Row():
            Button("Edit", variant="outline")
            Button("Delete", variant="destructive")
```
