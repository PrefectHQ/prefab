## Usage

Switches provide a visual toggle for binary states, offering an alternative to checkboxes.

<ComponentPreview auto json={`{"type":"Switch","label":"Enable notifications","checked":false,"size":"default","disabled":false,"required":false}`}>
<CodeGroup>
```python Python
from prefab_ui.components import Switch

Switch(label="Enable notifications")
```
```json Protocol
{
  "type": "Switch",
  "label": "Enable notifications",
  "checked": false,
  "size": "default",
  "disabled": false,
  "required": false
}
```
</CodeGroup>
</ComponentPreview>

## Sizes

Switches come in two sizes:

<ComponentPreview auto json={`{"type":"Column","gap":4,"children":[{"type":"Switch","label":"Default size","checked":true,"size":"default","disabled":false,"required":false},{"type":"Switch","label":"Small size","checked":true,"size":"sm","disabled":false,"required":false}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Switch,
)

with Column(gap=4):
    Switch(label="Default size", checked=True)
    Switch(label="Small size", size="sm", checked=True)
```
