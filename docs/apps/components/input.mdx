## With Labels

Pair inputs with labels for accessible forms:

<ComponentPreview auto json={`{"type":"Column","gap":2,"children":[{"type":"Label","text":"Email address"},{"type":"Input","inputType":"email","placeholder":"you@example.com","disabled":false,"required":true}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Input,
    Label,
)

with Column(gap=2):
```

## Date & Time Inputs

Native date and time pickers use the browser's built-in UI:

<ComponentPreview auto json={`{"type":"Row","gap":4,"children":[{"cssClass":"flex-1","type":"Column","gap":2,"children":[{"type":"Label","text":"Date"},{"type":"Input","inputType":"date","name":"date","disabled":false,"required":false}]},{"cssClass":"flex-1","type":"Column","gap":2,"children":[{"type":"Label","text":"Time"},{"type":"Input","inputType":"time","name":"time","disabled":false,"required":false}]},{"cssClass":"flex-1","type":"Column","gap":2,"children":[{"type":"Label","text":"Date & Time"},{"type":"Input","inputType":"datetime-local","name":"datetime","disabled":false,"required":false}]}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Input,
    Label,
    Row,
)

with Row(gap=4):
    with Column(gap=2, css_class="flex-1"):
        Label("Date")
        Input(input_type="date", name="date")
    with Column(gap=2, css_class="flex-1"):
        Label("Time")
        Input(input_type="time", name="time")
    with Column(gap=2, css_class="flex-1"):
        Label("Date & Time")
        Input(input_type="datetime-local", name="datetime")
```

## API Reference

<Card icon="code" title="Input Parameters">

<ParamField path="input_type" type="str" default="text">
  Input type: text, email, password, number, tel, url, search, date, time, datetime-local, file.
</ParamField>

<ParamField path="placeholder" type="str | None">
  Placeholder text. Supports `{{ field }}` interpolation.
</ParamField>

<ParamField path="value" type="str | None">
  Initial value. Supports `{{ field }}` interpolation.
</ParamField>

<ParamField path="name" type="str | None">
  Form field name for submission.
</ParamField>

<ParamField path="disabled" type="bool" default="False">
  Whether the input is disabled.
</ParamField>

<ParamField path="required" type="bool" default="False">
  Whether the input is required for form submission.
</ParamField>

<ParamField path="css_class" type="str | None">
  Additional CSS classes to apply.
</ParamField>
