## Usage

Switches provide a visual toggle for binary states, offering an alternative to checkboxes.

<ComponentPreview auto json={`{"type":"Switch","label":"Enable notifications","checked":false,"size":"default","disabled":false,"required":false}`} />

```python Switch Example
from prefab_ui.components import Switch

Switch(label="Enable notifications")
```

## Checked State

<ComponentPreview auto json={`{"type":"Switch","label":"Dark mode","checked":true,"size":"default","disabled":false,"required":false}`} />

```python Checked Switch
from prefab_ui.components import Switch

Switch(label="Dark mode", checked=True)
```

## Sizes

Switches come in two sizes:

<ComponentPreview auto json={`{"type":"Column","gap":4,"children":[{"type":"Switch","label":"Default size","checked":true,"size":"default","disabled":false,"required":false},{"type":"Switch","label":"Small size","checked":true,"size":"sm","disabled":false,"required":false}]}`} />

```python Switch Sizes
from prefab_ui.components import (
    Column,
    Switch,
)

with Column(gap=4):
    Switch(label="Default size", checked=True)
    Switch(label="Small size", size="sm", checked=True)
```

## Disabled State

<ComponentPreview auto json={`{"type":"Column","gap":3,"children":[{"type":"Switch","label":"Enabled switch","checked":false,"size":"default","disabled":false,"required":false},{"type":"Switch","label":"Disabled off","checked":false,"size":"default","disabled":true,"required":false},{"type":"Switch","label":"Disabled on","checked":true,"size":"default","disabled":true,"required":false}]}`} />

```python Disabled Switch
from prefab_ui.components import (
    Column,
    Switch,
)

with Column(gap=3):
    Switch(label="Enabled switch")
    Switch(label="Disabled off", disabled=True)
    Switch(label="Disabled on", checked=True, disabled=True)
```

## API Reference

<Card icon="code" title="Switch Parameters">

<ParamField path="label" type="str | None">

<ParamField path="checked" type="bool" default="False">
  Whether the switch is on.

<ParamField path="size" type="'sm' | 'default'" default="'default'">

<ParamField path="name" type="str | None">
  Form field name for submission.

<ParamField path="disabled" type="bool" default="False">
  Whether the switch is disabled.
</ParamField>

<ParamField path="required" type="bool" default="False">
