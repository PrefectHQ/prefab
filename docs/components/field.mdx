Field is a composable wrapper that handles both form validation and choice card layouts. It propagates `data-invalid` via CSS cascade, so labels turn red, controls get error styling, and error messages appear — all without wiring each control individually.

## Form Validation

Wrap a label, control, and optional error message in a Field. When `invalid=True`, all text descendants turn red automatically.

```python Python icon="python"
from prefab_ui.components import (
    Column, Field, FieldError, Input, Label, Select, SelectOption,
)

with Column(gap=4):
    with Field():
        Label("Username")
        Input(name="username", placeholder="Enter username")

    with Field(invalid=True):
        Label("Destination")
        with Select(placeholder="Choose a planet..."):
            SelectOption(value="mars", label="Mars")
            SelectOption(value="jupiter", label="Jupiter")
            SelectOption(value="saturn", label="Saturn")
        FieldError("Please select a destination.")
```

## API Reference

<Card icon="code" title="Field">
<ParamField body="invalid" type="bool" default="False">
  Whether the field is in an error state. Cascades red text to all children.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the field is dimmed and non-interactive.
</ParamField>

<ParamField body="orientation" type='"vertical" | "horizontal"' default='"vertical"'>
  Layout direction. `"vertical"` for form fields, `"horizontal"` for choice cards.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>

<Card icon="code" title="FieldTitle">
<ParamField body="content" type="str" required>
  Title text for the field.
</ParamField>
</Card>

<Card icon="code" title="FieldDescription">
<ParamField body="content" type="str" required>
  Description text for the field.
</ParamField>
</Card>

<Card icon="code" title="FieldContent">
  Container that groups FieldTitle and FieldDescription in horizontal layouts.
</Card>

<Card icon="code" title="FieldError">
<ParamField body="content" type="str" required>
  Error message text, always rendered in destructive (red) styling.
</ParamField>
</Card>
