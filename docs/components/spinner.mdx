Spinners indicate that content is loading or an action is in progress.

## Basic Usage

<ComponentPreview auto json={`{"view":{"type":"Spinner","size":"default"}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Spinner

Spinner()
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "type": "Spinner",
    "size": "default"
  }

```
</CodeGroup>
</ComponentPreview>

## Sizes

<ComponentPreview auto json={`{"view":{"cssClass":"gap-4 items-center","type":"Row","children":[{"type":"Spinner","size":"sm"},{"type":"Spinner","size":"default"},{"type":"Spinner","size":"lg"}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Row, Spinner

with Row(gap=4, css_class="items-center"):
    Spinner(size="sm")
    Spinner(size="default")
    Spinner(size="lg")
```

```json Protocol icon="brackets-curly"
{

        "type": "Spinner",
        "size": "lg"
      }

```
</CodeGroup>

## In Context

Pair a spinner with text to communicate what's loading.

<ComponentPreview auto json={`{"view":{"cssClass":"gap-2 items-center","type":"Row","children":[{"type":"Spinner","size":"sm"},{"type":"Text","content":"Computing the answer to life, the universe, and everything..."}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Row, Spinner, Text

with Row(gap=2, css_class="items-center"):
    Spinner(size="sm")
    Text("Computing the answer to life, the "
         "universe, and everything...")
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "cssClass": "gap-2 items-center",
    "type": "Row",
    "children": [
      {
        "type": "Spinner",
        "size": "sm"
      },
      {
        "type": "Text",
        "content": "Computing the answer to life, the universe, and everything..."
      }
    ]
  }
}
```
</CodeGroup>
</ComponentPreview>

## API Reference

<Card icon="code" title="Spinner Parameters">
<ParamField body="size" type="str" default="default">
  Spinner size: `"sm"`, `"default"`, `"lg"`.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>

## Protocol Reference

```json Spinner
{
  "type": "Spinner",
  "size?": "sm | default | lg",
  "cssClass?": "string"
}
```

For the complete protocol schema, see [Spinner](/protocol/spinner).
