## Usage

Textareas provide multi-line text input for comments, descriptions, and other longer content.

<ComponentPreview auto json={`{"type":"Textarea","placeholder":"Enter your message...","disabled":false,"required":false}`}>
<CodeGroup>
```python Python
from prefab_ui.components import Textarea

Textarea(placeholder="Enter your message...")
```
```json Protocol
{
  "type": "Textarea",
  "placeholder": "Enter your message...",
  "disabled": false,
  "required": false
}
```
</CodeGroup>

## With Labels

<ComponentPreview auto json={`{"type":"Column","gap":2,"children":[{"type":"Label","text":"Feedback"},{"type":"Textarea","placeholder":"Tell us what you think...","rows":5,"disabled":false,"required":false}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Label,
    Textarea,
)

with Column(gap=2):
    Label("Feedback")
    Textarea(
        placeholder="Tell us what you think...",
        rows=5,
    )
```
```json Protocol
{
  "type": "Column",
  "gap": 2,
  "children": [
    {
      "type": "Label",
      "text": "Feedback"
    },
    {
      "type": "Textarea",
      "placeholder": "Tell us what you think...",
      "rows": 5,
      "disabled": false,
      "required": false
    }
  ]
}
```
</CodeGroup>
</ComponentPreview>
