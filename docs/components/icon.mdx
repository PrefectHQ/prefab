Icons render any [lucide icon](https://lucide.dev/icons) by name. Pass the icon name in kebab-case and it resolves at render time.

## Basic Usage

<ComponentPreview auto json={`{"view":{"type":"Icon","name":"rocket","size":"default"}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Icon

Icon("rocket")
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "type": "Icon",
    "name": "rocket",
    "size": "default"
  }
}
```
</CodeGroup>
</ComponentPreview>

## Sizes

Three size variants control the icon dimensions.

<ComponentPreview auto json={`{"view":{"cssClass":"gap-4 items-center","type":"Row","children":[{"type":"Icon","name":"heart","size":"sm"},{"type":"Icon","name":"heart","size":"default"},{"type":"Icon","name":"heart","size":"lg"}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Icon, Row

with Row(gap=4, css_class="items-center"):
    Icon("heart", size="sm")
    Icon("heart", size="default")
    Icon("heart", size="lg")
```

```json Protocol icon="brackets-curly"
      }
    ]
  }
}
```
</CodeGroup>

## In Context

Icons work inline alongside text and other components.

<ComponentPreview auto json={`{"view":{"cssClass":"gap-3","type":"Column","children":[{"cssClass":"gap-2 items-center","type":"Row","children":[{"type":"Icon","name":"check","size":"sm"},{"type":"Text","content":"Task completed"}]},{"cssClass":"gap-2 items-center","type":"Row","children":[{"type":"Icon","name":"clock","size":"sm"},{"type":"Text","content":"Waiting for approval"}]},{"cssClass":"gap-2 items-center","type":"Row","children":[{"type":"Icon","name":"circle-x","size":"sm"},{"type":"Text","content":"Build failed"}]}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Icon,
    Row,
    Text,
)

with Column(gap=3):
    with Row(gap=2, css_class="items-center"):
        Icon("check", size="sm")
        Text("Task completed")
    with Row(gap=2, css_class="items-center"):
        Icon("clock", size="sm")
        Text("Waiting for approval")
    with Row(gap=2, css_class="items-center"):
        Icon("circle-x", size="sm")
        Text("Build failed")
```

## API Reference

<Card icon="code" title="Icon Parameters">
<ParamField body="name" type="str" required>
  Lucide icon name in kebab-case (e.g., `"arrow-right"`, `"circle-alert"`). Can be passed as a positional argument. Browse icons at [lucide.dev/icons](https://lucide.dev/icons).
</ParamField>

<ParamField body="size" type="str" default="default">
  Icon size: `"sm"`, `"default"`, `"lg"`.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>

## Protocol Reference

For the complete protocol schema, see the [Icon protocol definition](/protocol/icon).

```json Icon
{
  "type": "Icon",
  "name": "string (required)",
  "size?": "sm | default | lg",
  "cssClass?": "string",
  "visibleWhen?": "string"
}
```
