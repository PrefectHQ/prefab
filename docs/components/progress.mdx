Progress bars show how far along a process is. They fill from left to right as `value` approaches `max`.

## Basic Usage

<ComponentPreview json={{"view":{"type":"Progress","value":60.0,"min":0,"max":100,"variant":"default"}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Progress

Progress(value=60)
```
```json Protocol icon="brackets-curly"

  "view": {"type": "Progress", "value": 60.0, "min": 0, "max": 100, "variant": "default"}
}
```
</CodeGroup>
</ComponentPreview>

## Custom Max

By default `max` is 100, but you can set it to any positive number. The renderer normalizes the value to a percentage automatically.

<ComponentPreview json={{"view":{"cssClass":"gap-4","type":"Column","children":[{"cssClass":"gap-2","type":"Column","children":[{"type":"Label","text":"3 of 5 steps"},{"type":"Progress","value":3.0,"min":0,"max":5.0,"variant":"default"}]},{"cssClass":"gap-2","type":"Column","children":[{"type":"Label","text":"75%"},{"type":"Progress","value":75.0,"min":0,"max":100,"variant":"default"}]}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Column, Label, Progress

with Column(gap=4):
    with Column(gap=2):
        Label("3 of 5 steps")
        Progress(value=3, max=5)
    with Column(gap=2):
        Label("75%")
        Progress(value=75)
```

## Vertical Orientation

Set `orientation="vertical"` for vertical progress bars. The default height is set by the theme — wrap in a sized container for custom heights:

<ComponentPreview json={{"view":{"cssClass":"gap-8 h-48 items-end justify-center","type":"Row","children":[{"type":"Progress","value":75.0,"min":0,"max":100,"variant":"default","orientation":"vertical"},{"type":"Progress","value":50.0,"min":0,"max":100,"variant":"success","orientation":"vertical"},{"type":"Progress","value":25.0,"min":0,"max":100,"variant":"info","orientation":"vertical"}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Progress, Row
```

## Dynamic Progress with Labels

Bind a slider to a progress bar so the label reflects the actual value. The `percent` format specifier turns a raw fraction into a readable percentage:

<ComponentPreview json={{"view":{"cssClass":"gap-3","type":"Column","children":[{"cssClass":"justify-between items-center","type":"Row","children":[{"content":"Upload progress","type":"Text"},{"content":"{{ progress | percent }}","bold":true,"type":"Text"}]},{"type":"Progress","value":"{{ progress }}","min":0,"max":1.0,"variant":"default"},{"type":"Slider","min":0.0,"max":1.0,"value":0.65,"step":0.01,"name":"progress","disabled":false}]},"state":{"progress":0.65}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.app import set_initial_state
from prefab_ui.components import (
    Column,
    Progress,
    Row,
    Slider,
    Text,
)

set_initial_state(progress=0.65)

with Column(gap=3):
    with Row(css_class="justify-between items-center"):
        Text("Upload progress")
        Text("{{ progress | percent }}", bold=True)
    Progress(value="{{ progress }}", max=1)
    Slider(name="progress", min=0, max=1, step=0.01, value=0.65)
```

## API Reference

<Card icon="code" title="Progress Parameters">
<ParamField body="value" type="float | str" default="0">
  Current progress value. Can be a template expression for dynamic binding.
</ParamField>

<ParamField body="min" type="float" default="0">
  Minimum value. The renderer normalizes `(value - min) / (max - min)` to a 0–100 percentage.
</ParamField>

<ParamField body="max" type="float" default="100">
  Maximum value.
</ParamField>

<ParamField body="variant" type="str" default="default">
  Visual variant: `"default"`, `"success"`, `"warning"`, `"destructive"`, `"info"`, `"muted"`.
</ParamField>

<ParamField body="indicator_class" type="str | None" default="None">
  Tailwind classes for the indicator fill bar. Overrides variant coloring.
</ParamField>

<ParamField body="orientation" type="str" default="horizontal">
  Layout direction: `"horizontal"` or `"vertical"`.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Tailwind classes for the track (outer container).
</ParamField>
</Card>

```json Progress
}
```
