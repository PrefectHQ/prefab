Radio buttons let users select exactly one option from a set. Each `Radio` represents a single choice; wrap them in a `RadioGroup` to link them together and track the selected value.

## Basic Usage

A `RadioGroup` is a container that holds `Radio` children. Each radio needs a `value` (what gets stored when selected) and a `label` (what the user sees). The group tracks which radio is selected and stores its `value` string in state:

<ComponentPreview json={{"view":{"cssClass":"w-fit mx-auto","name":"radiogroup_1","type":"RadioGroup","children":[{"name":"radio_1","type":"Radio","value":"sm","label":"Small","checked":false,"disabled":false,"required":false},{"name":"radio_2","type":"Radio","value":"md","label":"Medium","checked":false,"disabled":false,"required":false},{"name":"radio_3","type":"Radio","value":"lg","label":"Large","checked":false,"disabled":false,"required":false}]}}}>

```python Python icon="python"
from prefab_ui.components import Radio, RadioGroup

    Radio(value="sm", label="Small")
    Radio(value="md", label="Medium")
    Radio(value="lg", label="Large")
```

## Reading the Value

Use `.rx` on the `RadioGroup` to get a reactive reference to the selected radio's `value` string. When the user clicks a different option, anything bound to `.rx` updates automatically:

<ComponentPreview json={{"view":{"cssClass":"gap-4 w-fit mx-auto","type":"Column","children":[{"name":"radiogroup_4","type":"RadioGroup","children":[{"name":"radio_10","type":"Radio","value":"heart-of-gold","label":"Heart of Gold","checked":false,"disabled":false,"required":false},{"name":"radio_11","type":"Radio","value":"bistromath","label":"Bistromath","checked":false,"disabled":false,"required":false},{"name":"radio_12","type":"Radio","value":"vogon-cruiser","label":"Vogon Constructor Fleet","checked":false,"disabled":false,"required":false}]},{"content":"Selected ship: {{ radiogroup_4 }}","type":"Text"}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Column, Radio, RadioGroup, Text

with Column(gap=4, css_class="w-fit mx-auto"):
    with RadioGroup() as group:
        Radio(value="heart-of-gold", label="Heart of Gold")
        Radio(value="bistromath", label="Bistromath")
        Radio(value="vogon-cruiser", label="Vogon Constructor Fleet")
    Text(f"Selected ship: {group.rx}")
```

Click a radio and the text updates instantly. The `.rx` reference holds the `value` string — `"heart-of-gold"`, `"bistromath"`, etc. — not the label.

## Actions

The `on_change` parameter on `RadioGroup` fires actions when the selection changes. Inside the action, `$event` holds the newly selected `value` string:

<ComponentPreview json={{"view":{"cssClass":"gap-4 w-fit mx-auto","type":"Column","children":[{"name":"radiogroup_5","type":"RadioGroup","onChange":{"action":"setState","key":"choice","value":"You picked: {{ $event }}"},"children":[{"name":"radio_13","type":"Radio","value":"light","label":"Light","checked":false,"disabled":false,"required":false},{"name":"radio_14","type":"Radio","value":"dark","label":"Dark","checked":false,"disabled":false,"required":false},{"name":"radio_15","type":"Radio","value":"system","label":"System","checked":true,"disabled":false,"required":false}]},{"cssClass":"text-muted-foreground","content":"{{ choice | default:'Make a selection' }}","type":"Text"}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Column, Radio, RadioGroup, Text
from prefab_ui.actions import SetState

with Column(gap=4, css_class="w-fit mx-auto"):
    with RadioGroup(
        on_change=SetState("choice", "You picked: {{ $event }}"),
    ):
        Radio(value="light", label="Light")
        Radio(value="dark", label="Dark")
        Radio(value="system", label="System", checked=True)
    Text(
        "{{ choice | default:'Make a selection' }}",
        css_class="text-muted-foreground",
    )
```

For simple "show the current selection" cases, `.rx` is the better approach. Use `on_change` when you need to transform the value or trigger side effects.

## API Reference

<Card icon="code" title="RadioGroup Parameters">
<ParamField body="name" type="str | None" default="None">
  State key for the selected value. Auto-generated if not provided.
</ParamField>

<ParamField body="on_change" type="Action | list[Action] | None" default="None">
  Action(s) to execute when the selection changes. `$event` is the `value` string of the selected radio.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>

<Card icon="code" title="Radio Parameters">
<ParamField body="value" type="str" required>
  The value stored in the group's state when this radio is selected.
</ParamField>

<ParamField body="label" type="str | None" default="None">
  Label text displayed next to the radio button.
</ParamField>

<ParamField body="checked" type="bool" default="False">
  Whether this radio is initially selected.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether this radio is non-interactive.
</ParamField>

<ParamField body="required" type="bool" default="False">
  Whether a selection is required for form submission.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
