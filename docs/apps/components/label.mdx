## Usage

Labels provide accessible text descriptions for form fields. They can contain text directly or wrap other components.

<ComponentPreview auto json={`{"type":"Label","text":"Email address"}`}>
<CodeGroup>
```python Python
from prefab_ui.components import Label

Label("Email address")
```
```json Protocol
{
  "type": "Label",
  "text": "Email address"
}
```
</CodeGroup>
</ComponentPreview>

## With Form Fields

Labels are typically paired with inputs and other form controls:

<ComponentPreview auto json={`{"type":"Column","gap":2,"children":[{"type":"Label","text":"Username","forId":"username"},{"type":"Input","inputType":"text","placeholder":"Enter username","name":"username","disabled":false,"required":false}]}`}>
<CodeGroup>
```python Python
from prefab_ui.components import (
    Column,
    Input,
    Label,
)

with Column(gap=2):
    Label("Username", for_id="username")
    Input(placeholder="Enter username", name="username")
```
```json Protocol
{
  "type": "Column",
  "gap": 2,

    }
  ]
}
```
</CodeGroup>
