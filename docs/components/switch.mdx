Switches provide a visual toggle for binary states, offering an alternative to checkboxes.

## Basic Usage

<ComponentPreview auto json={`{"$protocol":"prefab","view":{"type":"Switch","label":"Enable notifications","checked":false,"size":"default","disabled":false,"required":false}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Switch

Switch(label="Enable notifications")
```
```json Protocol icon="brackets-curly"
{
  "$protocol": "prefab",
  "view": {
    "type": "Switch",
    "label": "Enable notifications",
    "checked": false,
    "size": "default",
    "disabled": false,
    "required": false
  }
}
```
</CodeGroup>
</ComponentPreview>

## Sizes

Switches come in two sizes:

<ComponentPreview auto json={`{"$protocol":"prefab","view":{"cssClass":"gap-4","type":"Column","children":[{"type":"Switch","label":"Default size","checked":true,"size":"default","disabled":false,"required":false},{"type":"Switch","label":"Small size","checked":true,"size":"sm","disabled":false,"required":false}]}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Switch,
)

with Column(gap=4):
    Switch(label="Default size", checked=True)
    Switch(label="Small size", size="sm", checked=True)
```
```json Protocol icon="brackets-curly"
{
  "$protocol": "prefab",
  "view": {
    "cssClass": "gap-4",
    "type": "Column",
    "children": [
      {
        "type": "Switch",
        "label": "Default size",
        "checked": true,
        "size": "default",
        "disabled": false,
        "required": false
      },
      {
        "type": "Switch",
        "label": "Small size",
        "checked": true,
        "size": "sm",
        "disabled": false,
        "required": false
      }
    ]
  }
}
```
</CodeGroup>
</ComponentPreview>

## API Reference

<Card icon="code" title="Switch Parameters">

<ParamField body="label" type="str | None" default="None">
  Label text displayed next to the switch.
</ParamField>

<ParamField body="checked" type="bool" default="False">
  Whether the switch is on.
</ParamField>

<ParamField body="size" type="str" default="default">
  Switch size: `"default"`, `"sm"`.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  Form field name for submission.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the switch is non-interactive.
</ParamField>

<ParamField body="required" type="bool" default="False">
  Whether a selection is required for form submission.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>
