Badges are small, pill-shaped labels for status indicators, counts, and categories. They're inline elements that sit naturally alongside text or within cards.

```

from prefab_ui.components import Badge, Column, Row

with Column(gap=3):
    with Row(gap=2):
        Badge("Default")
        Badge("Secondary", variant="secondary")
        Badge("Destructive", variant="destructive")
        Badge("Outline", variant="outline")
    with Row(gap=2):
        Badge("Ghost", variant="ghost")
        Badge("Success", variant="success")
        Badge("Warning", variant="warning")
        Badge("Info", variant="info")
```

## Data Interpolation

Badge labels support `{{ field }}` placeholders:

```python Badge Interpolation
from prefab_ui.components import Badge

Badge("{{ status }}")
Badge("{{ count }} items", variant="secondary")
```
