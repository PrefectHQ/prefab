Sliders let users select a numeric value from a range by dragging a handle. The default range is 0–100.

## Basic Usage

<ComponentPreview auto json={`{"view":{"type":"Slider","min":0,"max":100,"value":50.0,"disabled":false}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Slider

Slider(value=50)
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "type": "Slider",
    "min": 0,
    "max": 100,
    "value": 50.0,
    "disabled": false
  }
}
```
</CodeGroup>

## Displaying the Value

Give the slider a `name` to store its current value in state, then reference it with `{{ }}` interpolation. Drag the slider and the label updates instantly:

<ComponentPreview auto json={`{"view":{"cssClass":"gap-2","type":"Column","children":[{"cssClass":"justify-between","type":"Row","children":[{"type":"Label","text":"Volume"},{"type":"Text","content":"{{ volume | number }}%","bold":true}]},{"type":"Slider","min":0,"max":100,"value":75.0,"name":"volume","disabled":false}]},"state":{"volume":75}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Label,
    Row,
    Slider,
    Text,
)

set_initial_state(volume=75)

with Column(gap=2):
    with Row(css_class="justify-between"):
        Label("Volume")
        Text("{{ volume | number }}%", bold=True)
    Slider(value=75, name="volume")
```
```json Protocol icon="brackets-curly"
{
  "view": {
    "cssClass": "gap-2",
    "type": "Column",
    "children": [
      {
        "cssClass": "justify-between",
        "type": "Row",
        "children": [
          {
            "type": "Label",
            "text": "Volume"
          },
          {
            "type": "Text",
            "content": "{{ volume | number }}%",
            "bold": true
          }
        ]
      },
      {
        "type": "Slider",
        "min": 0,
        "max": 100,
        "value": 75.0,
        "name": "volume",
        "disabled": false
      }
    ]
  },
  "state": {
    "volume": 75
  }
}
```
</CodeGroup>
</ComponentPreview>

The `| number` pipe strips trailing decimals so the display stays clean. This pattern works with any component that supports interpolation.

## Range and Step

Use `min`, `max`, and `step` to constrain the slider. Here the range is 0–10 with 0.5 increments:

<ComponentPreview auto json={`{"view":{"cssClass":"gap-2","type":"Column","children":[{"cssClass":"justify-between","type":"Row","children":[{"type":"Label","text":"Rating"},{"type":"Text","content":"{{ rating }}","bold":true}]},{"type":"Slider","min":0.0,"max":10.0,"value":5.0,"step":0.5,"name":"rating","disabled":false}]},"state":{"rating":5}}`}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import (
    Column,
    Label,
    Row,
    Slider,
    Text,
)

set_initial_state(rating=5)

with Column(gap=2):
    with Row(css_class="justify-between"):
        Label("Rating")
        Text("{{ rating }}", bold=True)
    Slider(value=5, name="rating", min=0, max=10, step=0.5)
```

## API Reference

<Card icon="code" title="Slider Parameters">

<ParamField body="value" type="float | None" default="None">
  Initial value of the slider.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  State key for the slider's current value. Set this to bind the slider to state for display or form submission.
</ParamField>

<ParamField body="min" type="float" default="0">
  Minimum value of the range.
</ParamField>

<ParamField body="max" type="float" default="100">
  Maximum value of the range.
</ParamField>

<ParamField body="step" type="float | None" default="None">
  Step increment. If not provided, the slider moves continuously.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the slider is non-interactive.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>
