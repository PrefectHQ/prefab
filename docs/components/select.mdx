Select dropdowns let users pick one option from a list.

## Basic Usage

```python Python icon="python"
from prefab_ui.components import (
    Select,
    SelectOption,

with Select(placeholder="Choose size..."):
```

## With Actions

The `on_change` parameter runs actions when the selection changes. `$event` is the `value` string of the selected option:

<ComponentPreview json={{"view":{"cssClass":"gap-4","type":"Column","children":[{"name":"select_6","type":"Select","placeholder":"Pick a color...","size":"default","disabled":false,"required":false,"onChange":{"action":"setState","key":"picked","value":"You chose: {{ $event }}"},"children":[{"type":"SelectOption","value":"red","label":"Red","selected":false,"disabled":false},{"type":"SelectOption","value":"green","label":"Green","selected":false,"disabled":false},{"type":"SelectOption","value":"blue","label":"Blue","selected":false,"disabled":false}]},{"cssClass":"text-muted-foreground","content":"{{ picked | default:'Make a selection' }}","type":"Text"}]}}}>
<CodeGroup>
```python Python icon="python"
from prefab_ui.components import Column, Select, SelectOption, Text
from prefab_ui.actions import SetState
from prefab_ui.rx import Rx

picked = Rx("picked")

with Column(gap=4):
    with Select(
        placeholder="Pick a color...",
        on_change=SetState("picked", f"You chose: {Rx('$event')}"),
    ):
        SelectOption(value="red", label="Red")
        SelectOption(value="green", label="Green")
        SelectOption(value="blue", label="Blue")
    Text(
        picked.default("Make a selection"),
        css_class="text-muted-foreground",
    )
```

## API Reference

<Card icon="code" title="Select Parameters">
<ParamField body="placeholder" type="str | None" default="None">
  Placeholder text shown when no option is selected.
</ParamField>

<ParamField body="name" type="str | None" default="None">
  State key for the selected value. Auto-generated if not provided. Use `.rx` to reference the current selection in other components without specifying the key explicitly.
</ParamField>

<ParamField body="size" type="str" default="default">
  Select size: `"default"`, `"sm"`.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether the select is non-interactive.
</ParamField>

<ParamField body="required" type="bool" default="False">
  Whether a selection is required for form submission.
</ParamField>

<ParamField body="on_change" type="Action | list[Action] | None" default="None">
  Action(s) to execute when the selection changes. `$event` is the `value` string of the selected option.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes appended to the component's built-in styles.
</ParamField>
</Card>

<Card icon="code" title="SelectOption Parameters">
<ParamField body="value" type="str" required>
  Option value for form submission.
</ParamField>

<ParamField body="label" type="str" required>
  Display text shown to the user.
</ParamField>

<ParamField body="selected" type="bool" default="False">
  Whether this option is initially selected.
</ParamField>

<ParamField body="disabled" type="bool" default="False">
  Whether this option is non-interactive.
</ParamField>

<ParamField body="css_class" type="str | None" default="None">
  Additional Tailwind CSS classes.
</ParamField>
</Card>
